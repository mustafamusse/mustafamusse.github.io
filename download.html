Easy guide for Mac:
Easy guide for Mac:
Extract the .zip file to your desktop.
Rename the .app to 'PyramidEscape.app'
Open the terminal
type "cd desktop" + enter
type "sudo chmod 777 PyramidEscape.app"
enter your password if prompted
type "cd PyramidEscape.app/Contents/MacOS/"
type "sudo chmod 777 PyramidEscape"
enter your password if prompted
Close terminal.
Try to open the .app file (it will not work yet)
Open Settings --> Security and integrity.
Allow the game to be executed
Start the .app file.
