Advanced guide for Mac:
In order for the .app to launch I had to set 'rwx' (maybe execute would be enough) permissions to both .app file and .app/Contents/MacOS/yourApp.
Navigate to folder with OSX.app 
in terminal write sudo chmod 777 OSX.app 
write cd OSX.app/Contents/MacOS 
write sudo chmod 777 EscapeRoom 
set computer to developer mode and allow external applications
